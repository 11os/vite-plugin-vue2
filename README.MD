# vite-plugin-vue2
**Current status: experimental**

Vite plugin for Vue2.

## Usage

``` js
// vite.config.js
module.exports = {
  plugins: [require('vite-plugin-vue2')]
}
```

## Supported Dev feature

- Dev Assets Resolve
- HMR
- Scoped Css
